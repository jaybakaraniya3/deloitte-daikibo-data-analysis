# 📊 Deloitte Daikibo Data Analytics Project

This project is part of a Data Analytics Job Simulation by Deloitte (via Forage), where real-world datasets were analyzed to generate business insights.

---

## 🚀 Project Overview

The objective of this project was to analyze operational and employee data from Daikibo factories to identify:

- Machine downtime patterns across global factory locations  
- High-risk machine types contributing to failures  
- Gender pay equality across different job roles  

---

## 📊 Project Tasks

### 🔹 Factory Downtime Analysis (Tableau)
- Built interactive dashboards to visualize machine downtime  
- Identified **Daikibo Shenzhen** as the factory with the highest downtime  
- Determined **Laser Cutter** as the most failure-prone machine  

### 🔹 Gender Pay Equality Analysis (Excel)
- Classified equality scores into:
  - Fair  
  - Unfair  
  - Highly Discriminative  
- Applied logical formulas to evaluate pay equity across roles  

---

## 🛠️ Tools & Technologies

- Tableau (Data Visualization)  
- Microsoft Excel (Data Analysis)  
- JSON Data Processing  

---

## 📌 Key Insights

- Shenzhen factory experienced the highest operational downtime  
- Certain machines contributed significantly to system failures  
- Pay inequality exists across specific job roles and requires attention
